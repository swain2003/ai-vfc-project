# 📌 Overview

AI-Integrated Vehicular Fog Computing (AI-VFC) is a research-oriented project focused on building an intelligent and low-latency Vehicular Fog Computing framework for Intelligent Transportation Systems (ITS).

The framework integrates advanced Artificial Intelligence techniques including:

- Deep Reinforcement Learning (DRL)
- Long Short-Term Memory (LSTM) Networks
- Federated Learning (FL)

The system is designed to optimize:

- Intelligent task offloading
- Mobility prediction
- Resource allocation
- Secure vehicular communication
- Low-latency edge computing

The proposed framework combines three computational layers:

- Vehicle Layer
- Fog Layer
- Cloud Layer

This project supports real-time vehicular communication, proactive handover management, distributed AI learning, and simulation-based performance evaluation.

# 🏗️ System Architecture

The AI-VFC framework follows a three-layer hierarchical architecture.

## 1️⃣ Vehicle Layer

- Collects vehicular sensor and mobility data
- Generates computation tasks
- Supports V2V, V2I, and V2F communication

## 2️⃣ Fog Layer

- Performs edge-side task offloading
- Applies AI-based optimization
- Handles queueing and handover-aware execution

## 3️⃣ Cloud Layer

- Processes high-latency workloads
- Performs global analytics
- Coordinates distributed model updates

---

# 🧠 AI Components

## 🔹 Deep Reinforcement Learning (DRL)

Used for intelligent task offloading decisions by minimizing latency and energy consumption.

## 🔹 LSTM Mobility Prediction

Predicts vehicle movement and RSU handovers proactively for seamless communication.

## 🔹 Federated Learning (FL)

Enables distributed model training without sharing raw vehicular data, improving privacy and scalability.

# ⚙️ Optimization Model

```text
C = αL + βE + γW
```

Where:

- **L** → Latency
- **E** → Energy Consumption
- **W** → Workload
- **α, β, γ** → Adaptive priority weights

# ▶️ How to Run the Project

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/swain2003/ai-vfc-project.git
cd ai-vfc-project
```

---

## 2️⃣ Create Virtual Environment

```bash
python -m venv .venv
```

---

## 3️⃣ Activate Virtual Environment

### Windows

```bash
.venv\Scripts\activate
```

### Linux / macOS

```bash
source .venv/bin/activate
```

---

## 4️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 5️⃣ Run the Simulation

```bash
python main.py
```

Generated outputs are stored inside:

```text
results/
```

Including:

- metrics.txt
- metrics.json
- simulation_output.csv
- plot images

---

## 6️⃣ Run the Dashboard

```bash
python dashboard/app.py
```

Open in browser:

```text
http://localhost:5000
```
